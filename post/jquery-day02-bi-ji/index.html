01-显示隐藏动画

  1. 默认显示和隐藏方式

    1. show([speed,[easing],[fn]])

      1. 参数：

        1. speed：动画的速度。三个预定义的值("slow","normal", "fast")或表示动画时长的毫秒数值(如：1000)

          1. easing：用来指定切换效果，默认是"swing"，可用参数"linear"

            • swing：动画执行时效果是 先慢，中间快，最后又慢
              • linear：动画执行时速度是匀速的
            1. fn：在动画完成时执行的函数，每个元素执行一次。
<script>
	// 2. hide([speed,[easing],[fn]])  隐藏
	//fn 回调函数(函数)
	//隐藏
	function hideFn(){
		$("#showDiv").hide(1000,"linear");
		
	}
	//显示
	function showFn() {
		$("#showDiv").show(1000,"linear");
	}
	//隐藏和显示
	function toggleFn(){
		$("#showDiv").toggle(1000,"linear");
	}

</script>

</head>
<body>
<input type="button" value="点击按钮隐藏div" onclick="hideFn()">
<input type="button" value="点击按钮显示div" onclick="showFn()">
<input type="button" value="点击按钮切换div显示和隐藏" onclick="toggleFn()">

<div id="showDiv" style="width:300px;height:300px;background:pink">
    div显示和隐藏
</div>
</body>

js的遍历方式

    <script type="text/javascript">
    	
    	 $(function(){
    	 	//js的便利
    	 	var cc= $("#city li");
    	 	
    	 	for(var i=0;i<cc.length;i++){
    	 		//js对象
    	 		//alert(cc[i].innerHTML); 
    	 		
    	 		//jquery对象获取值
    	 		alert($(cc[i]).html());	
    	 	}	 	
    	 })
    </script>

jq对象.each(callback)

带一个参数的

    <script type="text/javascript">
    	
    	 $(function(){
    	 	
    	 	//i表示索引
    	 $("li").each(function(i){
    	 	var cc=$(this).html();
    	 	alert("序号:"+(i+1)+"  ： "+cc);
    	 });	 	
    	 	
    	 })

    </script>

带两个参数的

    <script type="text/javascript">
    	
    	 $(function(){	
    	 $("li").each(function(index,vv){
    	 	//vv =this 是js对象
    	   alert(index+"  ：  "+$(vv).html());
    	 });	
    	 })
    </script>

回调函数

回调函数返回值：
				*如果当前function返回为false，则结束循环(break)。
				* 如果当前function返回为true，则结束本次循环，继续下次循环(continue)

$.each(object, [callback])

    <script type="text/javascript">
    	
    	 $(function(){
    	 		//参数1:遍历的对象 
    	 		//参数2:回调函数
    	 		var obj= $("li");
    	 		$.each(obj,function(){
    	 			var str= $(this).html();
    	 			alert(str);
    	 		});
    	 })
    </script>

for..of:

jquery 3.0 版本之后提供的方式
for(元素对象 of 容器对象)

    <script type="text/javascript">
    	
    	 $(function(){
    	 	//容器对象:需要遍历的对象或数组
    	 	//元素对象 接受遍历的变量
    	 	//for(元素对象 of 容器对象)
    	 	var str= $("li");
    	   for(cc of str){
    	   	alert($(cc).html());
    	   } 	
    	 })
    </script>
